DOSSIER IMAGES — Paray FC
=========================

Déposez ici le logo du club nommé exactement :

    logo-paray.webp

C'est le nom de fichier référencé par toutes les pages
(<img src="images/logo-paray.webp">), dans la barre de
navigation et dans le pied de page.

Tant que le fichier n'est pas présent, l'emplacement du logo
apparaîtra vide (texte alternatif « Logo Paray FC »).

Autres images à ajouter plus tard si besoin :
- photo d'ambiance du hero (page d'accueil)
- photos des actualités et des pôles

Pour l'instant, ces zones affichent un aplat « placeholder »
géré par la classe CSS .img-placeholder.
